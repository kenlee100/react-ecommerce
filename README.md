# react-ecommerce

### [Demo](https://kenlee100.github.io/react-ecommerce)

#### 功能

- 推薦空間
- 關於我們
- 購物車

#### 專案技術

- React 18.3.1
- Vite 6.0.5
- redux-toolkit 2.5.1
- react-router 7.1.5
- Tailwindcss 3
- SweetAlert2 11.7.2
- swiper 11.2.4
- SCSS

#### 安裝套件

```sh
pnpm install
```

#### 運行專案

```sh
pnpm run dev
```

#### 開啟專案

在瀏覽器網址列輸入以下即可看到畫面

```sh
http://localhost:3000/
```

#### 打包部署專案

```sh
pnpm run build
```
